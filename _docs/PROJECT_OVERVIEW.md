# SVU_Booking — Project Overview

## The Facility

**Swinburne's Virtual Universe (SVU)** is an immersive visualisation facility operated by Swinburne University of Technology, Melbourne, Australia.

### Hardware
- **Display**: 100m² curved LED wall providing an immersive panoramic environment
- **Audio**: 360° stereo audio system
- **3D / XR**: Full 3D capability; Unity-compatible for real-time interactive experiences
- **Capacity**: Groups ranging from school classes (~30) to invited public audiences

### Location
Swinburne University of Technology, Hawthorn Campus, Melbourne, Victoria, Australia.

---

## Vision

Build a fully **agentic AI-powered booking and ticketing platform** that handles every interaction — from a school teacher submitting a group visit request to an external company hiring the space for a corporate event — with minimal manual administration. The system should feel intelligent, proactive, and frictionless for all user types.

---

## Goals

1. **Eliminate manual admin overhead** — AI agents handle intake, scheduling, communications, and payments autonomously.
2. **Serve all user communities** — academics, school groups, public audiences, external hirers, and maintenance teams all have purpose-built flows.
3. **Maximise facility utilisation** — smart scheduling, conflict detection, and waitlist management keep the SVU in use.
4. **Provide operational insight** — real-time reporting on bookings, revenue, and utilisation for facility managers.
5. **Integrate with university systems** — Swinburne SSO for staff, university financial systems for invoicing.

---

## Use Cases

| Use Case | Description |
|----------|-------------|
| Academic Teaching | Academics book timeslots for lectures, tutorials, or research sessions |
| School / Education Outreach | Primary/secondary school groups visit for guided immersive sessions |
| Public Paid Events | Members of the public purchase tickets to curated SVU experiences |
| External Hire | Companies or organisations hire the full facility for private events |
| Maintenance | Facility technicians block time for hardware maintenance and calibration |
| Recurring Hire | Long-term clients set up weekly or monthly recurring block bookings |

---

## Operator Structure

- **Swinburne University** owns and operates the facility
- **Facility Manager(s)** act as Super Admins — full system control
- **Academic Staff** self-serve for teaching/research bookings
- **Operations Team** manages maintenance windows and technical issues
- A future phase may include a **public-facing event portal** branded as SVU Experiences
